## 项目简介
这是一个使用 **Qt** 框架、基于 **C++** 编写的 **明日方舟集成战略“探索者的银凇止境”** 比赛 **“可汗杯”** 的分数计算器。

## 项目启动
项目最好使用 **Qt Creator** 打开（打开`Rogue.pro`），本人使用的版本为：

```
Qt Creator 4.11.1
Based on Qt 5.14.1 (MSVC 2017, 32 bit)
```

最好不要用更低的版本启动。


## 发布版本
如果对源码不感兴趣，可以直接下载Release（暂只支持 **Windows** 系统）。
